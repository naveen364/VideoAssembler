Alternative: n8n → Railway FFmpeg (server-side, also free)

For fully automated video generation (no browser needed), deploy a free Flask + FFmpeg service on Railway. Your n8n workflow calls it via HTTP Request node and gets back a video URL.

Show Railway setup steps

Step 1 — Create a free account at railway.app (500 hrs/month free)

Step 2 — Deploy this one-click template: New Project → Deploy from GitHub → paste your repo with the Flask app below

Step 3 — Add an HTTP Request node at the end of your n8n workflow that POSTs your production_scenes JSON to your Railway URL

Step 4 — Railway returns a signed URL to the finished MP4 stored in a free Cloudflare R2 bucket

Flask app (app.py):

from flask import Flask, request, jsonify
import subprocess, os, uuid, requests, tempfile

app = Flask(__name__)

@app.route('/assemble', methods=['POST'])
def assemble():
    data = request.json
    scenes = data['production_scenes']
    color_filter = data.get('ffmpeg_color_filter',
        'eq=brightness=0.06:saturation=1.4:contrast=1.1')
    job_id = str(uuid.uuid4())[:8]
    tmpdir = tempfile.mkdtemp()
    clips = []

    for i, scene in enumerate(scenes):
        url = scene.get('video_url')
        if not url:
            continue
        out = f"{tmpdir}/clip_{i:02d}.mp4"
        # Download
        r = requests.get(url, timeout=30)
        raw = f"{tmpdir}/raw_{i:02d}.mp4"
        open(raw, 'wb').write(r.content)
        # Trim + grade
        duration = scene.get('duration_seconds', 8)
        overlay = scene.get('text_overlay', '')
        vf = color_filter
        if overlay:
            vf += f",drawtext=text='{overlay}':fontsize=52:fontcolor=white:x=(w-text_w)/2:y=h-120:shadowcolor=black:shadowx=2:shadowy=2"
        subprocess.run([
            'ffmpeg', '-y', '-i', raw,
            '-t', str(duration), '-vf', vf,
            '-c:v', 'libx264', '-preset', 'fast', out
        ], check=True, capture_output=True)
        clips.append(out)

    # Concat
    list_file = f"{tmpdir}/list.txt"
    with open(list_file, 'w') as f:
        for c in clips:
            f.write(f"file '{c}'\n")
    final = f"{tmpdir}/final_{job_id}.mp4"
    subprocess.run([
        'ffmpeg', '-y', '-f', 'concat', '-safe', '0',
        '-i', list_file, '-c', 'copy', final
    ], check=True, capture_output=True)

    # Upload to R2 / return base64 for small files
    with open(final, 'rb') as f:
        import base64
        b64 = base64.b64encode(f.read()).decode()
    return jsonify({'status': 'ok', 'video_base64': b64, 'job_id': job_id})

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=int(os.environ.get('PORT', 5000)))

Dockerfile:

FROM python:3.11-slim
RUN apt-get update && apt-get install -y ffmpeg && rm -rf /var/lib/apt/lists/*
WORKDIR /app
COPY requirements.txt .
RUN pip install flask requests
COPY app.py .
CMD ["python", "app.py"]

Then in n8n: HTTP Request → POST → https://your-app.railway.app/assemble → Body: your full JSON package
